ℹ️ Информация

Здесь вы можете редактировать оба типа приветственных сообщений:

✅ С подпиской (стартовое сообщение)

Сообщение, которое бот отправляет пользователям при команде /start, если они уже подписаны на канал.

  • HTML форматирование: Поддерживаются базовые HTML теги для стилизации текста
  • Медиа файлы: Можно добавить фото или видео с URL файла или использовать ID файла
  • Инлайн кнопки: Максимум 8 кнопок (ограничение Telegram)
  • Основные кнопки: "Ответи", "Приватный урок" и "Тарифы"

🚫 Без подписки

Сообщение, которое показывается пользователям, которые еще не подписались на канал.

  • HTML форматирование: Поддерживаются базовые HTML теги
  • Медиа файлы: Фото или видео (можно использовать URL файла или ID файла)
  • Посилання на канал: Для переходу користувача
  • ID каналу: Для перевірки підписки
  • Текст кнопки: Настраиваемый текст кнопки для перехода на канал

Принцип работы: Пользователь видит сообщение с кнопкой для подписки на канал. После подписки он может использовать команду /start для доступа к боту.

Важливо: Зміни застосовуються миттєво та зберігаються в SQLite базі даних.
