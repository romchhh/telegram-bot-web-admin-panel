ℹ️ Информация

ID пользователя: Уникальный идентификатор пользователя в Telegram

Имя пользователя: Username пользователя (если указан)

Дата регистрации: Когда пользователь впервые использовал бота

Последняя активность: Время последнего взаимодействия с ботом

Метка: Метка, с которой пользователь запустил бота (например: vid1, metka и т.д.)

Этап: Текущий этап прохождения пользователя в системе (с ID для удобства)

Статус подписки: Подписан ли пользователь на бота

Действия: Возможность изменить этап пользователя

Пагинация: На каждой странице отображается до {{ per_page }} пользователей

📋 Этапы прохождения:

1 - Нажал старт: Пользователь запустил бота

2 - Прошел капчу: Пользователь успешно прошел проверку

3 - Посмотрел ответы: Пользователь ознакомился с ответами

4 - Приватный урок: Пользователь просмотрел приватный урок

5 - Выбор тарифа: Пользователь ознакомился с тарифами

6 - Тарифы одежда: Пользователь посмотрел тарифы на одежду

7 - Тарифы техника: Пользователь посмотрел тарифы на технику

8 - Оплата техника: Пользователь нажал кнопку оплаты техники

9 - Оплата одежда: Пользователь нажал кнопку оплаты одежды

10 - Оплатил одежду: Пользователь успешно оплатил тариф на одежду

11 - Оплатил технику: Пользователь успешно оплатил тариф на технику
